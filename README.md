# 🌤 Weather App - Open-Meteo API

Una aplicación web moderna y responsiva que muestra el **clima actual** y el **pronóstico extendido** de cualquier ciudad utilizando la API gratuita de [Open-Meteo](https://open-meteo.com/).

Diseñada con HTML, CSS y JavaScript, esta app ofrece una experiencia visual atractiva con animaciones, íconos dinámicos y un **modo oscuro automático** que se activa según la hora del día.

 Características

- **Búsqueda por ciudad**: Ingresa el nombre de una ciudad y obtén su clima en tiempo real.
- **Clima actual**: Temperatura, velocidad del viento y hora local.
- **Pronóstico extendido**: Temperaturas máximas y mínimas para varios días.
-  **Frontend impactante**: Diseño moderno con tipografía elegante, glassmorphism y animaciones.
-  **Modo oscuro automático**: Se activa entre las 19:00 y las 6:00 para una experiencia nocturna más cómoda.
- **Responsivo**: Adaptado para móviles, tablets y pantallas grandes.

 Tecnologías utilizadas

- **HTML5**: Estructura semántica de la aplicación.
- **CSS3**: Estilos modernos, animaciones y diseño responsivo.
- **JavaScript (ES6+)**: Lógica de consumo de API y renderizado dinámico.
- **Font Awesome**: Íconos dinámicos para representar condiciones climáticas.
- **Open-Meteo API**: Fuente de datos meteorológicos gratuitos.
